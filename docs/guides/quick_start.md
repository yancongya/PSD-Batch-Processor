# 快速开始指南

## 第一步：安装依赖

### 方法 1：使用安装脚本（推荐）

双击运行 `install.bat`，它会自动：
- 检查 Python 版本
- 安装所需依赖
- 验证安装

### 方法 2：手动安装

```bash
pip install customtkinter pywin32
```

### 方法 3：使用 requirements.txt

```bash
pip install -r requirements.txt
```

## 第二步：验证环境（可选）

运行环境验证脚本，确保所有依赖正常：

```bash
python test_env.py
```

这会测试：
- ✅ customtkinter (GUI 框架)
- ✅ pywin32 (COM 控制)
- ✅ pathlib (路径处理)
- ✅ shutil (文件操作)
- ✅ concurrent.futures (线程池)

## 第三步：启动程序

```bash
python main.py
```

或者双击 `main.py` 文件（如果已关联 Python）。

## 第四步：配置设置

### 1. Photoshop 路径

- 默认路径：`C:\Program Files\Adobe\Adobe Photoshop 2025\Photoshop.exe`
- 如果路径不正确，点击"浏览..."选择正确的路径
- 如果 Photoshop 已运行，程序会自动连接

### 2. 脚本目录

- 默认：`scripts/`（程序同级目录）
- 点击"浏览..."选择其他目录
- 点击"刷新"按钮扫描所有 `.jsx` 文件

### 3. 备份目录

- 默认：`backups/`（程序同级目录）
- 每次处理会创建时间戳子文件夹
- 例如：`backups/20260123_184530/`

### 4. 选项设置

- **主题**：选择暗色或亮色主题
- **并发数**：建议选择 `1`（最稳定）
- **包含子文件夹**：根据需要勾选

### 5. 保存配置

点击"💾 保存配置"按钮，配置会保存到 `config.json`。

## 第五步：添加文件

### 添加单个/多个文件

1. 点击"添加文件"按钮
2. 选择 PSD 文件（可多选）
3. 文件会出现在列表中

### 添加整个文件夹

1. 点击"添加文件夹"按钮
2. 选择包含 PSD 的文件夹
3. 程序会自动扫描所有 PSD 文件
4. 如果勾选"包含子文件夹"，会递归扫描子目录

### 管理文件列表

- **移除选中**：选中文件后点击"移除选中"
- **清空列表**：点击"清空列表"清空所有文件
- **右键菜单**：右键点击文件可打开文件夹或用 Photoshop 打开

## 第六步：选择脚本

1. 在下拉框中选择要执行的 JSX 脚本
2. 可以点击"刷新"重新扫描脚本目录
3. 建议先用示例脚本测试：`example_flatten_image.jsx`

## 第七步：开始处理

1. 确认文件列表中有待处理的文件
2. 确认已选择正确的脚本
3. 点击"▶️ 开始处理"按钮
4. 弹出确认对话框，点击"是"

### 处理过程

程序会执行以下步骤：

1. **创建备份文件夹**：`backups/20260123_184530/`
2. **连接 Photoshop**：自动启动（如果未运行）
3. **逐个处理文件**：
   - 复制文件到备份目录
   - 打开 PSD 文件
   - 执行 JSX 脚本
   - 保存并关闭
   - 更新状态
4. **显示结果**：成功/失败统计

### 查看进度

- **进度条**：显示整体进度百分比
- **日志窗口**：实时显示处理日志（彩色）
- **文件列表**：每个文件的状态会实时更新

## 第八步：查看结果

### 处理完成

程序会弹出总结对话框：
- 成功数量
- 失败数量
- 总耗时

### 查看备份

点击"📂 打开备份文件夹"按钮，会自动打开最近的备份目录。

### 查看日志

日志会自动保存到 `logs/` 目录，文件名包含时间戳。

## 常见问题

### Q: 程序启动失败

**A**: 检查是否安装了依赖：
```bash
pip install customtkinter pywin32
```

### Q: 无法连接 Photoshop

**A**:
1. 检查 Photoshop 路径是否正确
2. 确保 Photoshop 已安装
3. 尝试手动启动 Photoshop 后再运行程序

### Q: 脚本执行失败

**A**:
1. 检查 JSX 语法是否正确
2. 在 Photoshop 中手动运行脚本测试
3. 查看日志窗口的错误信息
4. 确保脚本中有 `doc.save()` 调用

### Q: 备份文件在哪里？

**A**: 在配置的备份目录下，按时间戳命名的子文件夹中。点击"📂 打开备份文件夹"查看。

### Q: 如何修改主题？

**A**: 在界面右上角选择主题（暗色/亮色），然后点击"💾 保存配置"。
