## 🎯 首次使用步骤

### 1. 验证安装
```bash
python tests\test_fluent_v2.py
```
应该显示：`[SUCCESS] PyQt-Fluent-Widgets V2 版本测试通过！`

### 2. 启动应用
```bash
cd "F:\插件脚本开发\PSD Batch Processor\src"
python main_fluent.py
```

### 3. 配置 Photoshop 路径（必需）
**步骤**:
1. 点击导航栏的"⚙️ 设置"
2. 在"Photoshop 设置"卡片中点击"浏览..."
3. 选择 Photoshop.exe 文件
   - 通常路径：`C:\Program Files\Adobe\Adobe Photoshop 2025\Photoshop.exe`
4. 点击"保存设置"按钮

### 4. 刷新脚本列表
**步骤**:
1. 点击导航栏的"📝 脚本管理"
2. 点击"刷新脚本列表"按钮
3. 查看脚本列表，确认脚本已扫描

### 5. 添加文件并处理
**步骤**:
1. 点击导航栏的"🏠 文件处理"
2. 点击"添加文件"选择 PSD 文件
3. 或点击"添加文件夹"批量添加
4. 在"处理控制"中选择脚本
5. 调整并发数（可选）
6. 点击"开始处理"按钮

## 🔧 常见问题

### 问题 1: 模块导入错误
**错误**: `ModuleNotFoundError: No module named 'PyQtFluentWidgets'`

**解决**: 模块名是 `qfluentwidgets`，不是 `PyQtFluentWidgets`
```bash
# 检查导入
python -c "from qfluentwidgets import FluentWindow; print('OK')"
```

### 问题 2: 图标找不到
**错误**: `AttributeError: type object 'FluentIcon' has no attribute 'FOLDER_OPEN'`

**解决**: 已修复为 `FluentIcon.FOLDER`

### 问题 3: TreeWidgetItem 错误
**错误**: `ImportError: cannot import name 'TreeWidgetItem'`

**解决**: 已修复为 `QTreeWidgetItem` (来自 PyQt5.QtWidgets)

### 问题 4: GUI 不显示
**可能原因**: 在远程桌面或无图形界面环境中

**解决**:
- 在有图形界面的环境中运行
- 或使用 VNC/远程桌面

### 问题 5: 脚本列表为空
**原因**: 脚本目录不存在或没有 .jsx 文件

**解决**:
1. 检查"设置"页面的脚本目录路径
2. 确保 scripts/ 目录下有 .jsx 文件
3. 在"脚本管理"页面点击"刷新脚本列表"
