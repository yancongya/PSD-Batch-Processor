========================================
PSD Batch Processor - 文件清单
========================================

一、核心程序文件

1. main.py
   - 主程序入口
   - 依赖检查
   - 启动 GUI

2. test_env.py
   - 环境验证脚本
   - 测试所有依赖

3. quick_test.py
   - 快速测试脚本
   - 验证项目模块

4. install.bat
   - Windows 安装脚本
   - 自动安装依赖

5. requirements.txt
   - Python 依赖清单
   - customtkinter, pywin32

二、应用程序包 (app/)

1. app/__init__.py
   - 包初始化文件

2. app/ui/main_window.py
   - 主窗口界面（1200+ 行）
   - 设置区、文件列表、日志区、控制区
   - 主题切换、右键菜单

3. app/core/__init__.py
   - 核心包初始化

4. app/core/photoshop_controller.py
   - Photoshop COM 控制器（200 行）
   - 连接/断开、执行脚本、打开/关闭文档

5. app/core/processor.py
   - 批量处理器（350 行）
   - 备份、处理、进度回调、结果统计

6. app/models/__init__.py
   - 模型包初始化

7. app/models/file_item.py
   - 文件项数据模型（200 行）
   - FileItem、FileList、FileStatus

8. app/config/__init__.py
   - 配置包初始化

9. app/config/settings.py
   - 配置管理（180 行）
   - 读取/保存 JSON、路径验证

三、工具模块 (utils/)

1. utils/__init__.py
   - 工具包初始化

2. utils/logger.py
   - 日志工具（140 行）
   - 文件日志 + GUI 日志、彩色输出

四、示例脚本 (scripts/)

1. scripts/example_convert_to_grayscale.jsx
   - 灰度转换示例（10 行）

2. scripts/example_flatten_image.jsx
   - 压平图像示例（15 行）

3. scripts/example_resize_50_percent.jsx
   - 缩小 50% 示例（15 行）

五、文档 (docs/)

1. docs/todo.md
   - 需求文档（200 行）
   - 项目概述、功能需求、技术栈、架构

2. docs/quick_start.md
   - 快速开始指南（300 行）
   - 安装、配置、使用流程

3. docs/project_structure.md
   - 项目结构说明（250 行）
   - 目录详解、模块说明、数据流

4. docs/development_summary.md
   - 开发总结（400 行）
   - 完成情况、代码统计、测试情况

5. docs/project_overview.md
   - 项目概览（350 行）
   - 功能清单、技术栈、常见问题

6. docs/checklist.md
   - 完成清单（300 行）
   - 详细检查项、验证结果

六、配置和说明文件

1. README.md
   - 项目主文档（300 行）
   - 功能特性、使用说明、示例

2. .gitignore
   - Git 忽略配置
   - 排除 logs、backups、config.json 等

3. 使用说明.txt
   - 中文使用指南（150 行）
   - 快速上手说明

4. 文件清单.txt
   - 本文件
   - 项目文件列表

七、运行时目录（自动生成）

1. backups/
   - 备份文件存储
   - 按时间戳创建子文件夹

2. logs/
   - 日志文件存储
   - 按日期生成日志文件

3. config.json（首次运行后生成）
   - 用户配置
   - Photoshop 路径、脚本目录等

========================================
如何开始使用
========================================

1. 安装依赖
   - 双击 install.bat
   - 或运行: pip install customtkinter pywin32

2. 快速测试
   - 运行: python quick_test.py
   - 验证所有模块是否正常

3. 启动程序
   - 运行: python main.py
   - 或双击 main.py

4. 查看文档
   - README.md - 完整文档
   - 使用说明.txt - 快速指南
   - docs/quick_start.md - 详细步骤
